# Contributing to NBA Quant Engine

Thank you for your interest in contributing! This project is a quantitative analytics engine for NBA player prop predictions.

## 🚀 Getting Started

1. Fork the repository
2. Clone your fork locally
3. Create a virtual environment: `python3 -m venv venv && source venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Copy `.env.example` to `.env` and add your API keys
6. Run the full pipeline (see README for steps)

## 📋 Guidelines

### Code Style
- Follow PEP 8 conventions
- Use type hints where practical
- Add docstrings to all public functions

### Pull Requests
- Create a feature branch: `git checkout -b feature/your-feature`
- Write clear commit messages
- Ensure all scripts run without errors before submitting
- Update the README if you add new features or scripts

### Feature Ideas
- Additional data sources (e.g., play-by-play data, advanced stats)
- New ML architectures (e.g., neural networks, transformers)
- Dashboard / web UI for viewing predictions
- Slack / Telegram notification integration
- Historical odds data for more accurate backtesting

## 🐛 Bug Reports

Please open an issue with:
- A clear description of the bug
- Steps to reproduce
- Expected vs. actual behavior
- Python version and OS
